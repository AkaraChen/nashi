# package-starter

Opinionated TypeScript + esbuild Starter Template

## Features

Fast by default, easy to use, batteries included.

1. ESM/CJS bundler.
2. pnpm, esbuild, born with fastness.
3. unit test with Vitest.
4. TypeScript of course.

## Use

### build

```shell
pnpm run build
```

### lint

```shell
pnpm run lint
```

### test

```shell
pnpm test
```
